Projects

  • Project 1: MUNFridgeGUI

    Duration: Three Months
    Technologies Used: Node.js, HTML, CSS, JavaScript, TypeScript, Docker

    ● Designing the GUI for a smart fridge application.
    ● Working with a group of 4 for the GUI and 21 people for the overall application.
    ● Utilizing Node.js to develop a 3-D render of a smart fridge with UI capabilities.
    ● This allows the user to store the groceries in the refrigerator and keep track of items as well as all information stored by the user.
    ● Utilizing Docker Containers, we were able to communicate with the other teams which included: Messaging Team, Database Team, ALgorithms Team

  • Project 2: Teleportal

    Duration: Two Months
    Technologies Used: Unity Game Engine, Visual Studio, C#

    ● Developed a game using the Unity Game Engine.
    ● The Portal videogame franchise influenced the game, which has a portal gun allowing users to travel around the level.
    ● Designed different levels with puzzles that rely on strategic problem-solving skills.

  • Project 3: E-commerce Console

    Duration: Two Months
    Technologies Used: Visual Studio, C++

    ● Developed a visual studio project which allowed the user to enter data through the console.
    ● The user could add items, price, quantity, and category of item to help track their items.
    ● They could also put in information for different users if another user wanted to make their list.
    ● This project allowed the user to check the prices of the items and sort them with the lowest price, category, or quantity.
